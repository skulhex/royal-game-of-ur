# Royal Game of Ur
Это реализация древней настольной игры **Royal Game of Ur** на Python.
Игра была обнаружена в Уре, Месопотамия, и насчитывает около 5000 лет.

В этой версии можно играть вдвоём в консоли с визуализацией доски и фишек.

## Правила игры
- Каждый игрок имеет 7 фишек.
- Цель — довести все фишки до финиша (Goal).  
- Игроки бросают четыре "костяные монеты", каждая решка даёт 1 шаг(всего 0-4 шагов).
- Фишки движутся по пути (`PATH_X` и `PATH_O`).
- На розетках фишка защищена от выбивания и даёт дополнительный ход.
- Чтобы попасть в Goal, нужен точный бросок.
- Если фишка попадает на клетку противника (не розетку), она выбивает противника обратно в Home.

## Запуск
```bash
git clone https://github.com/skulhex/royal_game_of_ur.git
cd royal_game_of_ur
python3 main.py
```

## Как играть
- В начале хода автоматически выбрасываются кости
- После броска отображаются доступные ходы.
- Игра продолжается по очереди, пока один из игроков не доведёт все фишки до Goal.
